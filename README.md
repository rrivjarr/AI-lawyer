# 📜 Nyaya AI – Your Legal Companion

An AI-powered LegalTech platform that provides intelligent legal assistance, document analysis, and automated legal document generation using modern web technologies and AI.

---

## 🚀 Overview

**Nyaya AI** is a smart legal assistant designed to simplify legal processes for everyday users. It leverages AI to provide legal guidance, analyze documents, and generate agreements, all within a user-friendly interface.

---

## 🛠 Tech Stack

* **Frontend:** React (TypeScript), Vite
* **Styling:** Tailwind CSS, Tailwind Typography
* **AI Integration:** Google Gemini AI
* **Libraries:**

  * react-markdown (for rendering AI responses)
  * jsPDF (for PDF generation)

---

## ✨ Features

* 🤖 **AI Legal Chat** – Ask legal questions and get AI-powered answers
* 📄 **Document Analyzer** – Analyze and summarize legal documents
* 📝 **NDA Builder** – Generate and download NDAs as PDF
* 👨‍⚖️ **Lawyer Directory** – Browse legal professionals
* 📅 **Reminder Calendar** – Track legal deadlines
* 📰 **Legal Updates** – Stay updated with legal news
* 🌍 **Multilingual Support** – Supports multiple languages

---

## 🧠 How It Works

1. User interacts with the React-based UI
2. Input is sent to the AI service layer
3. Prompts are processed and sent to Gemini AI
4. AI generates a response
5. Output is rendered using Markdown or exported as PDF

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/nyaya-ai.git

# Navigate to project folder
cd nyaya-ai

# Install dependencies
npm install

# Run development server
npm run dev
```

## ⚙️ Environment Setup

Create a `.env` file and add your API key:

